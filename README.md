# House of Smith - Point of Sales & Inventory System

Aplikasi berbasis web sederhana untuk memanajemen inventori dan transaksi kasir (POS), dikembangkan menggunakan HTML, CSS, dan Vanilla JavaScript. Data disimpan secara real-time menggunakan Local Storage.

## Fitur Utama (IPO)
- **Input**: Form tambah barang baru, Form update stok masuk.
- **Proses**: Kalkulasi transaksi kasir, Validasi unit testing, Pengurangan/Penambahan stok otomatis.
- **Output**: Tabel pemantauan (Monitoring) stok barang real-time, Cetak Struk Kasir.

## Cara Menjalankan Aplikasi
1. Clone repositori ini: `git clone <link-repo>`
2. Buka folder `src/`
3. Buka file `index.html` menggunakan browser apa saja (Chrome/Edge/Firefox).
4. Gunakan kredensial berikut untuk login:
   - **Username**: admin
   - **Password**: admin123

## Cara Menjalankan Unit Testing
1. Buka folder `tests/`
2. Buka file `runner.html` menggunakan browser.
3. Laporan pengujian otomatis (Pass/Fail) akan langsung ditampilkan di layar.
